## 📋 À propos

**ITALYCARE 360** est un service d'accompagnement 360° pour les étrangers qui souhaitent vivre, investir, étudier ou prendre leur retraite en Italie.

Nous couvrons **18 domaines de services** :

- 🏠 Immobilier — achat, vente, location
- 🛠️ Rénovation et travaux
- 🏥 Suivi médical et système de santé
- 📦 Export Made in Italy
- 🎓 Visa étudiant et inscriptions universitaires
- ⚖️ Juridique et fiscal
- 🛎️ Conciergerie premium
- 📄 Administratif (codice fiscale, permis de séjour)
- 💍 Organisation d'événements (mariages)
- 🏡 Maisons de retraite
- 💰 Financement et mutui
- 🏗️ Professionnels certifiés (architectes, avocats, médecins)
- 👵 Silver Economy — pension en Italie (régime 7%)
- 🔑 Property Care — gestion de biens à distance
- 🌡️ Cures thermales
- ✈️ Aviation privée
- 📚 Scolarité pour enfants expatriés
- 🚗 Véhicules — import et immatriculation

- **Frontend** : Next.js 14 (App Router, static export)

- **Backend API** : Cloudflare Worker (chat, leads, traduction)
- **Email transactionnel** : Resend (domaine vérifié `italycare360.com`)
- **Hébergement site** : GitHub Pages
